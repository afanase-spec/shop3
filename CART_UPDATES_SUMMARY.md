## 🚀 Как использовать

### Для пользователя:

1. **Добавление товара со страницы товара**:
   - Нажимаете кнопку с ценой (например "89,90 ₽ +")
   - Открывается модальное окно
   - Выбираете количество кнопками +/- или вручную
   - Видите итоговую сумму сразу
   - Нажимаете "Добавить в корзину"
   - Товар добавлен!

2. **Просмотр суммы корзины**:
   - Смотрите в правый верхний угол (header)
   - Под иконкой корзины видите сумму
   - Сумма обновляется в реальном времени
   - Работает на всех страницах

### Для разработчика:

**Добавление товара через JavaScript**:
```javascript
// Старый способ (все еще работает)

// Функция автоматически обновит:
// - Badge с количеством
```

## 🔧 Технические детали

### Форматирование цены в JavaScript:
```javascript
total.toLocaleString('ru-RU', {
    minimumFractionDigits: 2,
    maximumFractionDigits: 2
}) + ' ₽'
// Результат: "1 234,56 ₽"
```

### Структура модального окна:
```
#quantityModal
├── .modal-header (градиентный фон)
│   ├── Заголовок "Выберите количество"
│   └── Кнопка закрытия
├── .modal-body
│   ├── Название товара
│   ├── Цена за единицу
│   ├── Контролы +/-
│   └── Итоговая сумма
└── .modal-footer
    ├── Кнопка "Отмена"
    └── Кнопка "Добавить в корзину"
```

### Обновление суммы в header:
1. AJAX запрос к API
2. Получаем `data.cartTotal`
3. Вызываем `updateCartBadge(count, total)`
4. Функция находит `.cart-link`
5. Создает/обновляет `.cart-total-badge`
6. Форматирует и отображает сумму
