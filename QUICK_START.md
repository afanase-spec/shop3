# 🚀 Быстрый старт - Обновление страницы товара

## ⚡ Что нужно сделать СЕЙЧАС (3 шага):

### Шаг 1: Обновите базу данных (ОБЯЗАТЕЛЬНО!)

Откройте phpMyAdmin → http://localhost/phpmyadmin

1. Выберите базу данных `shop2` (слева)
2. Нажмите вкладку **SQL** (вверху)
3. Скопируйте и вставьте этот код:

```sql
ALTER TABLE `products` 
ADD COLUMN `manufacturer` varchar(200) DEFAULT NULL AFTER `is_popular`,
ADD COLUMN `composition` text DEFAULT NULL AFTER `manufacturer`,
ADD COLUMN `calories` decimal(6,2) DEFAULT NULL AFTER `composition`,
ADD COLUMN `proteins` decimal(6,2) DEFAULT NULL AFTER `calories`,
ADD COLUMN `fats` decimal(6,2) DEFAULT NULL AFTER `proteins`,
ADD COLUMN `carbohydrates` decimal(6,2) DEFAULT NULL AFTER `fats`;
```

4. Нажмите кнопку **"Вперед"** (Go) внизу
5. Должно появиться сообщение "Запрос успешно выполнен" ✅

---

### Шаг 2: Протестируйте админ-панель

1. Войдите в админ-панель: http://localhost/shop2/admin/
2. Перейдите в **Товары**
3. Нажмите **"Добавить товар"** или **"Редактировать"** любой существующий товар
4. Прокрутите вниз - вы должны увидеть новые разделы:
   - 📦 **Дополнительная информация** (производитель, состав)
   - 📊 **Пищевая ценность (на 100г)** (калории, белки, жиры, углеводы)

5. Заполните поля для тестового товара:
   - Производитель: `Домик в деревне`
   - Состав: `Молоко цельное, витамины A и D`
   - Калории: `62.00`
   - Белки: `3.20`
   - Жиры: `3.20`
   - Углеводы: `4.70`

6. Нажмите **"Сохранить"**

---

### Шаг 3: Проверьте страницу товара

1. Откройте каталог: http://localhost/shop2/catalog.php
2. Нажмите на товар, который вы только что обновили
3. На странице товара вы должны увидеть:
   - ✅ Производителя под названием
   - ✅ Карточку с составом
   - ✅ Красивую таблицу БЖУ с фиолетовыми значениями
   - ✅ Овальную кнопку с ценой (фиолетовый градиент)

4. Наведите курсор на кнопку - должна быть анимация
5. Нажмите на кнопку - должен появиться ripple эффект (волны)

---

## 🎯 Готово!

Теперь вы можете:
- ✅ Добавлять производителя и состав для любых товаров
- ✅ Заполнять БЖУ для продуктов питания
- ✅ Оставлять БЖУ пустым для бытовой химии (таблица не покажется)
- ✅ Все изменения сразу видны на странице товара

---

## 💡 Советы:

### Для продуктов питания:
Заполняйте все поля БЖУ - появится красивая таблица на странице товара

### Для бытовой химии/непищевых товаров:
Оставляйте поля БЖУ пустыми - таблица не будет отображаться

### Редактирование:
Любые изменения в админ-панели сразу отражаются на сайте - не нужно ничего обновлять дополнительно!

---

## ❓ Если что-то не работает:

1. **Ошибка SQL при выполнении ALTER TABLE**:
   - Возможно, колонки уже существуют
   - Проверьте структуру таблицы products в phpMyAdmin
   
2. **Новые поля не видны в админ-панели**:
   - Обновите страницу (Ctrl+F5)
   - Очистите кэш браузера

3. **Таблица БЖУ не показывается на странице товара**:
   - Убедитесь, что заполнено поле "Калории"
   - Таблица появляется ТОЛЬКО если калории > 0

4. **Кнопка не имеет фиолетовый градиент**:
   - Обновите страницу (Ctrl+F5)
   - Проверьте, что файл custom.css подключен
