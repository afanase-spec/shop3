# Сайт доставки продуктов

Полноценный сайт доставки продуктов и бытовых товаров, разработанный на PHP + MySQL с современным дизайном.

## Функциональные возможности

### Публичная часть:
- Главная страница с популярными товарами и категориями
- Каталог товаров с фильтрацией и поиском
- Карточка товара с подробным описанием
- Корзина с AJAX обновлением (без перезагрузки страницы)
- Оформление заказа с валидацией данных
- Страница контактов с информацией о доставке

### Административная панель:
- Дашборд со статистикой
- Управление товарами (добавление, редактирование, удаление)
- Управление категориями
- Управление заказами (просмотр, изменение статуса)
- Фильтрация заказов по статусу и дате

## Технологии

- **Backend:** PHP 7.4+ (без фреймворков, чистая архитектура)
- **Database:** MySQL с PDO (подготовленные запросы)
- **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript (ванильный)
- **Icons:** Font Awesome 6
- **Fonts:** Google Fonts (Inter)

## Требования

- PHP 7.4 или выше
- MySQL 5.7 или выше
- Включенные расширения: pdo_mysql, session
- OpenServer (или другой локальный сервер)

## Установка

### Шаг 1: Размещение файлов

Поместите папку `shop2` в директорию доменов OpenServer:
```
C:\OSPanel\domains\shop2
```

### Шаг 2: Создание базы данных

1. Запустите OpenServer
2. Откройте phpMyAdmin (обычно http://localhost/tools/phpmyadmin/)
3. Создайте новую базу данных с именем `delivery`:
   - Кодировка: `utf8mb4_unicode_ci`

### Шаг 3: Импорт SQL дампа

1. В phpMyAdmin выберите базу данных `delivery`
2. Перейдите на вкладку "Импорт"
3. Выберите файл `install.sql` из корня проекта
4. Нажмите "Вперед" для импорта

### Шаг 4: Проверка конфигурации

Откройте файл `includes/config.php` и проверьте настройки:

```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'delivery');
define('DB_USER', 'root');
define('DB_PASS', '');
define('SITE_URL', 'http://shop2');
```

Если ваш домен отличается от `shop2`, измените `SITE_URL`.

### Шаг 5: Запуск сайта

Откройте браузер и перейдите по адресу:
```
http://shop2/
```

## Структура базы данных

### Таблицы:

1. **categories** - Категории товаров
   - id, name, slug, created_at

2. **products** - Товары
   - id, category_id, name, slug, description, price, image, is_popular, created_at

3. **orders** - Заказы
   - id, customer_name, phone, address, comment, status, total, created_at

4. **order_items** - Элементы заказа
   - id, order_id, product_id, quantity, price_at_time

5. **admins** - Администраторы
   - id, username, password_hash, created_at

## Безопасность

Проект включает защиту от основных уязвимостей:

- **SQL-инъекции:** Использование подготовленных запросов PDO
- **XSS:** Экранирование всех выводимых данных через htmlspecialchars()
- **CSRF:** Токены для всех форм
- **Пароли:** Хеширование через password_hash()

## Возможные проблемы и решения

### Ошибка подключения к БД
**Решение:** Проверьте настройки в `includes/config.php` (хост, имя БД, пользователь, пароль)

### Не работают сессии
**Решение:** Проверьте права на запись в временную папку PHP

### Не загружаются изображения
**Решение:** Проверьте правильность путей к изображениям в базе данных

### Белый экран
**Решение:** Включите отображение ошибок в PHP (display_errors = On) для диагностики

### Ошибка 404 на страницах
**Решение:** Убедитесь, что домен настроен правильно в OpenServer и SITE_URL в config.php
