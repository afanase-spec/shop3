## 📊 Технические детали

### Использованные CSS техники:

1. **Cubic-bezier transitions**: Для более естественных движений
   ```css
   transition: all 0.4s cubic-bezier(0.175, 0.885, 0.32, 1.275);
   ```

2. **Transform комбинации**: Одновременное использование translate и scale
   ```css
   transform: translateY(-8px) scale(1.02);
   ```

3. **Keyframe animations**: Множество кастомных анимаций
   - fadeIn, fadeInUp, fadeInDown, fadeInLeft, fadeInRight
   - scaleIn, pulse, float, slideInRight, badgeBump, spin

4. **Hover эффекты**: Улучшенные состояния наведения
   - Изменение теней
   - Трансформации
   - Цветовые переходы

5. **Active states**: Эффекты нажатия
   - Scale down для ощущения клика

### Производительность:

Все анимации используют:
- `transform` и `opacity` (GPU-accelerated свойства)
- `will-change` где необходимо
- Оптимизированные timing functions
- Отсутствие layout thrashing

- Круглые элементы: `50%`
